# 📸 Photo Booth — Releases

This repo hosts the installer and auto-update files for the **Photo Booth** kiosk application.

## 📥 Download

Head to the [Releases](https://github.com/clean-code-id/optik150k-photo-booth-releases/releases) page to download the latest Windows installer.

## 🔄 Auto-Update

The app checks this repo for updates on launch. When a new version is available, it downloads in the background and installs on next app quit.

## 🎯 About

A fullscreen photo booth kiosk application built with Electron — pick a layout, snap photos, apply frames, and print. ✨
